# PakShield AI Backend

AI-powered Pakistani currency authenticity detection backend.

## API

- `GET /api/health` — Health check
- `POST /api/detect` — Upload banknote image for analysis
- `GET /api/history` — Scan history
- `POST /api/chat` — Chat assistant
- `GET /api/currency/*` — Currency info endpoints
